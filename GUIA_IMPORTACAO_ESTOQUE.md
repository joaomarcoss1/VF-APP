# Guia de Importação de Estoque

A rota `/importacao/estoque` aceita XLSX e CSV para criar/atualizar produtos e estoque. Campos esperados: Produto, código de barras, SKU, categoria, quantidade, valor de custo, valor de venda e estoque mínimo.

PDF textual é aceito apenas quando a tabela é selecionável. Para PDFs em imagem, converta para XLSX/CSV.
