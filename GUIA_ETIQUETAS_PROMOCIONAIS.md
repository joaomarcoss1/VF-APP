# Guia de Etiquetas Promocionais

A rota `/etiquetas` permite criar etiquetas comuns e promocionais. Modelos disponíveis: padrão claro, promoção relâmpago, promoção elegante, institucional, minimalista, somente logo, QR Code e térmica preto e branco.

## Promoção relâmpago
1. Escolha o modelo “Promoção relâmpago”.
2. Defina título, subtítulo e data final.
3. Informe preço promocional no produto selecionado.
4. Gere prévia ou salve o lote.
5. Imprima em A4, térmica ou exporte ZPL.
