# Guia Admin Master

O Admin Master opera somente pela área `/master`.

Funções:

- Criar empresa.
- Gerar código/matrícula.
- Vincular Admin da Empresa.
- Ver status de plano.
- Auditar acesso.
- Ver suporte e saúde do SaaS.

O Admin Master não deve usar o painel comum de empresa para operar dados de clientes, salvo fluxo auditado.
