# Guia de Teste Etiquetas e Scanner

- Criar produto com código.
- Gerar etiqueta simples.
- Gerar etiqueta promocional.
- Gerar etiqueta com paleta da empresa.
- Imprimir A4 e térmica.
- Ler etiqueta pelo scanner.
- Confirmar abertura do produto correto.
- Confirmar que etiqueta de outra empresa não abre produto.
